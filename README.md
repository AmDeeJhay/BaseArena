# 🎮 BaseArena — Skill-Based Competition & Earning Platform

BaseArena is a gamified, onchain platform where users compete with their skills, prove their abilities, and earn rewards.

It transforms skill-building from a passive process into a live, social, and competitive experience, combining challenges, leaderboards, and rewards into one continuous loop — all powered onchain by Base.

## 🚀 Why BaseArena?

In today's digital world:

- Skills are hard to verify
- Platforms are fragmented (learn here, compete there, earn elsewhere)
- Most Web3 apps focus on earning without engagement

BaseArena solves this by unifying everything into one system:

- 🎮 Compete through real-world challenges
- 🧠 Prove your skills with measurable outcomes
- 🏆 Rank on leaderboards
- 💰 Earn rewards based on performance
- 👥 Connect with other builders socially

## 🧠 Core Idea

**Play. Prove. Earn.**

BaseArena is not just a platform — it's an Arena where:

- skills are tested
- performance is visible
- and value is rewarded

## 🛠️ Core Features

| Feature | Description |
|---------|-------------|
| 🎮 Arena Challenges | Users participate in skill-based challenges across categories (coding, design, writing, etc.) |
| 🏆 Leaderboards | Real-time rankings based on performance, XP, and achievements |
| 💰 Onchain Rewards | Earn rewards from challenges, bounties, and competitions |
| 🧠 Skill Verification | Submissions are evaluated via AI + human validation |
| 👤 User Profiles | Public skill identity with XP, badges, and achievements |
| 🤝 Social Layer | Follow, connect, and interact with other builders |
| 🎯 Bounties | Companies or individuals post real-world tasks with rewards |
| 🎮 Mini Games (Optional) | Skill-based games for engagement and micro-rewards |

## 🧑‍💼 User Roles

- **Competitor** – Participates in challenges and earns rewards
- **Challenger** – Posts challenges or bounties
- **Verifier** – Reviews and validates submissions (AI + human)
- **Community Member** – Engages socially, follows, and collaborates

## 🔗 Onchain Integration (Base)

BaseArena leverages Base for:

- 💰 Reward distribution (challenges, bounties, competitions)
- 🔐 Wallet-based identity and authentication
- 🧾 Proof of participation and achievement
- ⚡ Fast, low-cost micro-transactions

## ⚙️ Tech Stack

| Layer | Tool |
|-------|------|
| Blockchain | Base (Ethereum L2) |
| Smart Contracts | Solidity |
| Backend | Nest.js |
| Database | PostgreSQL |
| Frontend | Next.js + Tailwind CSS |
| AI | OpenAI (evaluation + feedback systems) |
| Hosting | Vercel (Frontend), Render (Backend) |
